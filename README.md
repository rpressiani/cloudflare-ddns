# Cloudflare Dynamic DNS Updater

[![Docker Build & Push](https://github.com/rpressiani/cloudflare-ddns/actions/workflows/main.yml/badge.svg)](https://github.com/rpressiani/cloudflare-ddns/actions/workflows/main.yml)

A simple Go application that updates a Cloudflare DNS record with your current public IP address. Perfect for home servers or dynamic IP connections.
